# News Classifier Project

Проект за класификация на новинарски статии с използване на машинно обучение.

## Технологии
- Python
- scikit-learn
- TF-IDF
- Naive Bayes
- Паралелна обработка (ThreadPool + Pipeline)

## Данни
- Над 1000 новинарски статии
- Категории: sport, politics, technology

## Резултати
- Точност: 1.00

## Функционалности
- Автоматична класификация на текст
- Паралелна обработка
- Запис на резултатите във файл
